## ne7
"Not Exactly 7" is a terminal text coloring utility

## Usage
```
   --file [file_name]   - Select the file and print output with mixed RGB colors
   --text [message]     - Text message to colorize
   --ms [milliseconds]  - Wait for provided milliseconds on animation before recolorize (default: 1000ms)
   --sec [seconds]      - Wait for provided seconds on animation before recolorize (default: 1s)
   --amin               - Enable/disable animation of text buffer
   --freq [frequency]   - Set RGB frequency (default: 0.3)
   --seed [limit]       - Set a seed limit (default: 0..255, random)
   --spread [limit]     - Set a spread limit (default: 8.0)
   --noise [limit]      - Set a noise limit (default: 7)
   --showcur            - Make the terminal cursor visible
   --hidecur            - Make the terminal cursor hidden
```
You can get this help screen by running `ne7 --help`
